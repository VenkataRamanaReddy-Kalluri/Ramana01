# HR-Analytics-Dashboard (Interactive Dashboard Creation Using Excel)

# Project Objective
To analyze HR data in Excel and identify key trends in employee attrition, performance, and demographics to support effective HR decision-making.

# process
- Collect HR data Clean and prepare data, 
- Remove errors/duplicates, fix date and text formats
- Load data then Import the cleaned Excel/CSV data
- Create KPIs and charts
- Add filters and design layout

## Final Conclusion
This HR analysis dashboard in Excel gives a clear, data‑driven view of employee attrition by showing overall headcount, attrition, and active employees, then breaking attrition down by gender, age, education, department, job role, and marital status so HR can quickly identify high‑risk groups and trends and design targeted retention actions instead of relying on guesswork.
